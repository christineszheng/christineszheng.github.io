What is a system of equation?

Intro to System of Equations Page

A system of equations is a set of two or more equations that share the same variables. A solution to a system of equations is a set of numbers that make all of the equations true. In other words, the solution will satisfy every equation in the system. There are various methods to solve a system of equations that we will go over on this website. These methods include substitution, elimination, and graphing. On a graph, the solution to a system of equations is the point of intersection. Visually, you look for the point where the lines touch each other.

What is the substitution method?

In order to solve a system of equations using substitution, one of the equations must have an isolated variable. As the name implies, you substitute the expression into the corresponding variable in your other equation. The purpose of using substitution is to end of with an equation that only has one variable.


How to solve a system of equations using substitution method:

Step 1:Rewrite one of the equation in y= or x=
Step 2:Substitute into the other equation.
Step 3:Solve for your variable.
Step 4:Substitute again to solve for your 2nd variable.
Practice:
y=6x-11
-2x-3y=-7

What is the elimination method?

The elimination method requires you to add equations in order to eliminate or “cancel out” a variable. This helps create a simpler equation with one variable to help solve. However, in order to add equations, you must first create inverse coefficients.

Steps to solve a system of equations using elimination:

Step 1:Line up your variables (your equations should be in standard form).
Step 2:Multiply one or more equations to create equivalent equations with an inverse coefficient.
Step 3:Add the equations and solve
Step 4:Substitute back into one of the original equations to solve for the other variable. Practice:
-4x-2y=-12
4x+8y=-24
